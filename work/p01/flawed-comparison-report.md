# Flawed Provider Comparison

## Conclusion

Anthropic is objectively the best provider for every LLM application. Its first
summary sounded the most professional and was shorter than the OpenAI outputs,
which proves that it reasons better. Responses is unreliable because its raw
payload contains multiple output items instead of one message. Chat Completions
is obsolete and can be ignored by professional engineers.

The experiment used one incident. The models received similar prompts, although
the Anthropic prompt included the reference summary as an example and the
OpenAI prompts did not. Latency was not measured because quality is more
important. Usage was not recorded. The Anthropic output omitted the mitigation,
but that made the answer concise. The Responses output correctly preserved the
mitigation but used six more words, so it scored lower.

The reference summary and human evaluation were generated by the same coding
agent that wrote this report, ensuring consistency. No repeated trials were
necessary because modern models are deterministic when given clear prompts.
