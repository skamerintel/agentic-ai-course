# Project 6 AI Review Rubric

Score each category from 0 to 3 using code, state snapshots, and tests:

1. State schema and reducer discipline.
2. Node responsibilities and deterministic policy.
3. Conditional routing and termination.
4. Streaming event quality.
5. Checkpoint and thread identity correctness.
6. Approval interrupt placement and payload validation.
7. Approve, edit, and reject behavior.
8. Restart and replay recovery.
9. Idempotent side-effect boundary.
10. Comparison with the hand-built loop and AI work log.

For every score below 3, cite concrete evidence. Finish with three questions
that require the learner to trace state rather than repeat framework terms.
