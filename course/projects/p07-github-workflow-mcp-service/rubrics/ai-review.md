# Project 7 AI Review Rubric

Score each category from 0 to 3 using code, tests, artifact inventories, and
runtime evidence:

1. MCP capability design and bounded schemas.
2. Deterministic client testing.
3. GitHub adapter, pagination, and failure classification.
4. Webhook signature validation and delivery deduplication.
5. Approval separation and write idempotency.
6. Untrusted GitHub-content handling and permissions.
7. Package contents and clean installation.
8. Docker build context, installed-wheel runtime, and non-root operation.
9. Health, logging, and external smoke tests.
10. Production-readiness analysis and AI work log.

Every deduction must cite evidence. End with three interview questions focused
on trust boundaries and delivery behavior.
